# SCSS 固有規約

CSS/SCSS 共通の規約については [`css.md`](./css.md) を参照してください。

このファイルでは SCSS 固有の規約のみを定義します。

---

## ネスティング

- ネストは最大 3 階層まで
- `&` セレクタは BEM の Element/Modifier に使用する
- `@media` はブロック（セレクタ）内にネストする（コンポーネントのスタイルを一箇所にまとめる）

  ```scss
  .block {
    width: 100%;

    @media (min-width: 768px) {
      width: 50%;
    }
  }
  ```

## 変数

- 変数名は `$kebab-case` を使用する
- グローバル変数は専用ファイルで管理する
- マジックナンバーを変数化する

## ミックスイン・関数

- 再利用可能なスタイルはミックスインにする
- ミックスインは単一責務にする
- `@extend` より `@mixin` を優先する（出力 CSS の予測可能性のため）

## ファイル構成

- パーシャルファイルは `_` プレフィックスをつける
- 機能ごとにファイルを分割する
