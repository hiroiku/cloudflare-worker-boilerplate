# CSS/SCSS 共通規約

このファイルは CSS と SCSS で共通のスタイリング規約を定義します。

SCSS 固有の規約については [`scss.md`](./scss.md) を参照してください。

---

## セレクタ

- ID セレクタ（`#id`）をスタイリングに使用しない。JavaScript のフックにのみ使用する
- セレクタのネストは最大 3 階層までに抑える
- ユニバーサルセレクタ（`*`）の使用は避ける
- 要素セレクタよりクラスセレクタを優先する

## 命名規約

- クラス名は kebab-case を使用する
- BEM（Block Element Modifier）記法を推奨する
- 意味のある命名をし、見た目ではなく役割を表す（`red-text` ではなく `error-message`）

## プロパティ

- プロパティはアルファベット順または論理的なグループ順で記述する
- ショートハンドプロパティを適切に使用する
- `0` の単位は省略する（`0px` ではなく `0`）
- 色は小文字の 16 進数または CSS 変数を使用する

## フォーマット

- インデントは 2 スペース
- セレクタと `{` の間にスペースを入れる
- プロパティの `:` の後にスペースを入れる
- 各プロパティは別の行に記述する

## パフォーマンス

- `!important` の使用は避ける
- 過度に詳細なセレクタを避ける
- 未使用の CSS を削除する
- CSS カスタムプロパティ（変数）を活用する
